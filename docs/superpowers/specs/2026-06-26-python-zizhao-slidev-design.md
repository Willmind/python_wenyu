# Python 自招两小时 Slidev 设计

## 目标

制作一套可运行的 Slidev 课件，帮助已有 P01-P06 基础的初中生在两小时内完成自招 Python 上机题复盘、变式练习和开放题表达准备。

## 范围

课件只围绕 P01-P06 图片中的知识点展开：

- P01：`turtle`、`import`、`for range`
- P02：`input()`、`float()`、变量、面积公式
- P03：`if ... else ...`
- P04：圆面积和周长公式转代码
- P05：`if / elif / else`
- P06：循环累加 `s = s + i`

二维码、打卡、投票、小程序只作为开放题/面试表达素材，不要求学生现场掌握第三方库。

## 教学策略

主线是“看注释补代码”，不是系统性 Python 科普。每个知识点按同一节奏讲：

1. 识别题型。
2. 套固定模板。
3. 找常见扣分点。
4. 做一个同类变式。
5. 转成校园项目表达。

## 交付

- Slidev 项目骨架。
- 中文 `slides.md` 主课件。
- 自定义样式。
- 原始 P01-P06 图片作为附录资产。
